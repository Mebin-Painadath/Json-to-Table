# Json to Table #
======================

## prerequisite ##
Python 3.7
SQL Server

## Steps to follow ##
Step 1:- Change to directory to "DjangoTable".
Step 2:- Install dependencies from dependencies.txt
Step 3:- Execute commands
  i) python manage.py migrate
  ii) python manage.py runserver
